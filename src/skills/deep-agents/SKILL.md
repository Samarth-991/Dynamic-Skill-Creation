# Deep Agents

## Overview
Deep Agents is an opinionated, ready-to-run agent harness for building AI agents and LLM applications. It provides a customizable agent that can plan, read/write files, manage context, and execute tasks with various tools and models.

## Automatic Processing
The skill automatically processes user queries or task descriptions to generate agent responses or task results.

## Core Capabilities
- Planning and task management
- Filesystem access and file I/O
- Shell execution and process management
- Sub-agents and multi-agent support
- Smart defaults and context management
- Customizable tools and models
- LangChain integration

## Workflow
1. **Initialization**: The skill initializes the Deep Agents framework and loads the necessary libraries and tools.
2. **User Input**: The user provides a query or task description, which is passed to the agent.
3. **Planning**: The agent plans the task and determines the required tools and models.
4. **Execution**: The agent executes the task using the selected tools and models.
5. **Context Management**: The agent manages context and updates the task state.
6. **Response Generation**: The agent generates a response or result based on the task execution.

## Usage Patterns
### Pattern 1: Basic
- User query: "Research LangGraph and write a summary"
- Agent response: A summary of LangGraph

### Pattern 2: Advanced
- User query: "Create a report on the current state of AI research using multiple agents and tools"
- Agent response: A comprehensive report generated by multiple agents and tools
